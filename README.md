## **Project Description**

Kanban Pro is a web-based Kanban task management application that allows users to create multiple boards, add tasks, edit tasks, delete tasks, and move tasks between boards using drag-and-drop. All data is stored locally in the browser using localStorage, making the app fully functional without an external database or server.

Live Demo: https://olabisi546.github.io/Kanban-Application/

## **Features**

* Create, rename, and delete boards.  
* Add, edit, and delete tasks within boards.  
* Drag-and-drop tasks between boards.  
* Task status indicators: Overdue, Due Soon, On Track, No Due Date.  
* Local data persistence with localStorage.

### **Overall Style & Theme**

The design aims for a clean, minimalistic interface that keeps the focus on task management. Boards are displayed horizontally for easy overview, and tasks are presented with clear labels and status indicators.

### **Color Palette**

* Primary: Shades of red to highlight important actions such as creating or saving tasks.  
* Background: Light gradients for a soft and approachable look.  
* Text: Darker reds and maroons for headings and important labels.  
* Buttons: Contrasting colours for actionable buttons and hover effects.

## **How to Run**

1. Download and extract the ZIP file.  
2. Ensure all files are in the same directory.  
3. Open index.html in a web browser (Live Server is recommended for full functionality).

The application should load immediately without additional setup, and all functionality will work locally.
